Tashia NPC Mod versions 2.05C and below were written by Michael Lyashenko
(sentinelsx@yahoo.com) in April 2002, while versions above "C" and up
to "F" were released by Maus a year later to be more compatible with the
latest mods, among other improvements. The updated version includes the
original English romance, French translation option by Yoho, German
translation by Maus, Spanish Translation by Clan DLAN, and Italian translation by Dario (rtt-mod@libero.it). Latest released version is 2.05G.
